I'm an aspiring web developer currently looking for a junior position. My fascination with web development started in 2007 buidling basic web sites using HTML and CSS. I recently started to learn JavaScript based on how amazed I am by React and what it can do to improve user interfaces and experiences.

My goal is to develop web applications that will assist not-for-profit organizations. Currently I'm building an application that will automate the language assistance protocol process for an affordable housing agency. If you're interested in working with me then please contact me.
